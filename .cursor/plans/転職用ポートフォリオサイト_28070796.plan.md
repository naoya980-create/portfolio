# 転職用ポートフォリオサイト作成計画

## 技術構成

- **HTML5**: セマンティックなマークアップ（`header`, `main`, `section`, `footer`）
- **CSS3**: レスポンシブ・印刷しやすいスタイル、変数で色・フォントを一元管理
- **JavaScript**: 必要最小限（スムーススクロール、モバイル用ナビなど）

## サイト構成

```
portfolio/
├── index.html      # 単一ページ（4セクションすべて含む）
├── css/
│   └── style.css   # メインスタイル
└── js/
    └── main.js     # 軽いインタラクション用
```

## ページ構成（1ページ内のセクション）

1. **ヘッダー / ナビ**
  - サイトタイトル（例：「〇〇のポートフォリオ」）
  - アンカーリンク: 自己紹介 / 学歴 / 職歴 / 取得資格
2. **自己紹介**
  - 見出し + プロフィール文（名前・一言・スキルなどのプレースホルダー）
  - 写真用エリアは `img` の `alt` とプレースホルダー用コメントで対応
3. **学歴**
  - 年〜年、学校名・学部学科などのリスト形式（2〜3件のプレースホルダー）
4. **職歴**
  - 期間、会社名、役職・業務内容のリスト形式（2〜3件のプレースホルダー）
5. **取得資格**
  - 取得年、資格名のリスト形式（2〜3件のプレースホルダー）
6. **フッター**
  - コピーライト、連絡先プレースホルダー（メール等）

## デザイン方針

- **トーン**: 転職向けで清潔感のあるレイアウト（白基調 + アクセント色1色）
- **フォント**: システムフォントまたは Web フォント（Google Fonts の Noto Sans JP 等）で可読性を優先
- **レスポンシブ**: モバイルではハンバーガーメニュー、タブレット以上では横並びナビ
- **印刷**: `@media print` でナビ非表示・背景色なしなど、履歴書と一緒に印刷しやすい指定

## プレースホルダー内容

- 日本語で「【ここに〇〇を入力】」形式のコメントと、差し替えしやすい短いサンプル文を配置
- `index.html` 内で検索しやすいよう、セクションごとに `id="profile"`, `id="education"`, `id="career"`, `id="certifications"` を付与

## 実装時の注意

- 外部依存は極力減らす（CDN の Google Fonts のみ想定、オフラインでも崩れないように）
- アクセシビリティ: 見出し階層（h1 → h2）の一貫、リンク・ボタンのフォーカス表示を考慮
- ファイルはすべて `portfolio` 直下またはその配下に作成し、相対パスで参照

## 成果物

- ブラウザで `index.html` を開くだけで表示できる静的サイト
- 後から本人が名前・学歴・職歴・資格・連絡先を書き換えやすいシンプルな HTML と CSS のみの構成
